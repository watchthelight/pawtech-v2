## Review System

| Feature | Description | Rejection Reason |
|---------|-------------|------------------|
| Template responses | Pre-built rejection reasons with one click | Declined by owner |
| Bulk actions | Batch reject left-server/spam apps | Declined by owner |
| Review queue dashboard | `/review queue` with priority sorting | Declined by owner |
| Escalation system | Flag apps for senior mod review | Declined by owner |
| Auto-reject rules | Auto-reject left server, blank answers | Declined by owner |
| Internal notes | Mods leave comments for handoffs | Declined by owner |
